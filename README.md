# 개요
- Alomst 쇼핑몰 웹 페이지
- React를 사용하여 회원가입, 상품리스트, 장바구니 기능 등의 페이지를 반응형으로 구현하였습니다.

# 기술 환경
- Front : HTML5/CSS3, JavaScript, Bootstrap 
- Back : Node.js

# 프로젝트 설명
- 반응형 웹페이지
- 쇼핑몰 웹 서비스
- Ajax 요청으로 상품 리스트 출력
